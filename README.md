# Randlang language support extension
This is the randlang language support extension, which belongs to randlang/[The Randlang Compiler](https://github.com/RK4444/RandlangCompiler). You can use it, by copying it to the extensions folder of vscode on your operating system e.g.
```
cp -r <randlang> ~/.vscode/extensions
```
then reload or reopen vscode and start using randlang.
## State of the project
Currently, only syntax highlighting is supported, but snippets are planned.
